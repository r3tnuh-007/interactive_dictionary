# interactive_dictionary
This is an interactive dictionary

***It takes a word as an input, search at the data source or knowledge base and shows you the meaning of the word***
***OBS:***_The data stored in the data source is totally in english_

# Requirements
***1 - Have Python3 install***
<br>
_You can do that executing the command ***"sudo apt install python3"***_
<br><br>
***2 - Execute the code***
<br>
_You can do that executing the command ***"python3 dictionary.py"***_
<br><br>
***3 - final***
<br>
_You can put any word as an input to know its meaning_
